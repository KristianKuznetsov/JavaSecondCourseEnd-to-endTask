Данное приложение читает файл, находит в нем математические выражения,
вычислеет их и записывает в файл с указанным именем

Типами поддерживаемых фалов являются:
".json", ".xml", ".txt"

В связи с недоработками
поддерживаемыми операциями являютс:
"+", "-", "/", "*" "()"

Программа способна распознавать следующие виды чисел:
целые, с плавающей точкой, в экспоненциальной форме
Числа с плавающей точкой можно указывать как с точкой, так и с запятой

В связи с недоработкой
для корректных вычислений обязательно указывать знак умножения

Программа не распознает (не поддерживает)
любые функции (sqrt, cos, sin, ...)
символы ("^", "|", "$", ... )
операторы равенства ("=", ">", "<", ...)
!!! для операции деления необходимо использовать "/"
!!! операция ":" не поддерживается

Для записи полученного результата в файл в
нижнем полле ввода напишини название файла в который
бует сохранен результат

Примечание при указании расширения
программа создаст файл с этим расширением, без указания
или при использовании неизвестного расширения
произойдет замена на расширение по умолчанию (".txt")

После сохранения файла, с помощью соответствующих кнопок
его можно заархивировать и зашифровать
